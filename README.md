# CUSTOM BIKE FILE Ver.3

CUSTOM BIKE FILEの画面・操作確認用プロトタイプです。

## Ver.3で追加した内容
- 「ノーマル車も大歓迎」の案内
- ノーマル車／カスタム途中／フルカスタムの投稿選択
- 状態別ラベルと絞り込み検索
- バイク詳細ページ
- 写真ギャラリー（最初の12枚＋もっと見る）
- お気に入り（ブラウザ内に保存）
- マイページ
- 編集・写真追加・売ります・SOLD操作の画面

## 起動方法
1. Node.jsをインストール
2. このフォルダで `npm install`
3. `npm run dev`
4. ブラウザで `http://localhost:3000`
